# FAS Landing
_ Just a status page for Force Apps / Services _


## Commands / Installation
### Node Stuff
```sh
# Install Modules
npm install

# Build site for docker image
npm run build
```

### Docker Stuff
```sh
# Run the contianer
docker-compose up -d
```
